# Blog Content Structure Examples

## Module 1: Daily Stock Pick (宝藏标的)

### Template

```
【美股每日宝藏标的 | YYYY/MM/DD】

股票代码 + 公司全称：XXX / Company Name
所属行业 + 核心主营业务：Industry / Core Business

✅ 核心价值亮点（3句话以内）
• 亮点1: 详细描述...
• 亮点2: 详细描述...
• 亮点3: 详细描述...

⚠️ 核心风险提示（2句话以内）
• 风险1: 详细描述...
• 风险2: 详细描述...

💡 长期持有建议（1句话）
• 持有周期X-Y年，仓位不超过组合的Z%，适合作为XXX板块的配置标的
```

### Example: VST (Vistra Corp)

```
【美股每日宝藏标的 | 2026/03/05】

股票代码 + 公司全称：VST / Vistra Corp.
所属行业 + 核心主营业务：电力公用事业 / 竞争性电力生产与零售

✅ 核心价值亮点
• AI电力需求爆发受益者：美国AI数据中心电力需求呈指数级增长，Vistra作为德州最大电力供应商占据核心区位优势，未来5年数据中心电力需求预计翻倍
• 战略并购驱动增长：公司持续通过并购扩张版图，近期完成对Energy Harbor核电资产的收购，核电作为基荷电源在AI时代价值重估
• 估值存在安全边际：相比同业龙头NextEra Energy（NEE）的估值溢价，Vistra当前估值仍处于合理偏低区间

⚠️ 核心风险提示
• 区域市场波动风险：电力价格受德州ERCOT市场供需波动影响显著，极端天气事件可能造成短期业绩剧烈波动
• 监管政策不确定性：核电业务面临严格的联邦和州级监管，任何核安全事故都可能冲击公司长期盈利能力

💡 长期持有建议
• 建议持有周期2-4年，单个标的仓位控制在投资组合的3%-5%以内，Vistra适合作为能源基础设施板块的卫星配置
```

## Module 2: Investment Wisdom (投资认知加餐)

### Template

```
【每日投资认知加餐 | YYYY/MM/DD | 投资风格】

核心主题：一句话点明核心概念

核心理念：
投资大师/经典著作的核心观点，详细阐述...

可落地执行规则：
1. 规则1：具体操作...
2. 规则2：具体操作...
3. 规则3：具体操作...

经典案例极简解读：
100字以内案例描述 + 核心逻辑 + 可借鉴经验

一句话记忆点：
提炼核心结论
```

### Example: Margin of Safety

```
【每日投资认知加餐 | 2026/03/05 | 价值投资】

核心主题：安全边际的本质——以4毛钱买1块钱的东西

核心理念：
巴菲特在历年《致股东的信》中反复强调："投资的第一原则是永远不要亏损，第二原则是永远记住第一原则。"这一看似简单的表述，实则是对格雷厄姆在《聪明的投资者》中提出的安全边际（Margin of Safety）概念的终极诠释。

可落地执行规则：
1. 计算内在价值：使用DCF（现金流折现）模型，对企业未来现金流进行保守估计。关键在"保守"——宁可低估也不高估
2. 设定买入阈值：只有当市场价格 ≤ 内在价值 × 0.6（即6折以下）时才考虑建仓
3. 分批建仓纪律：首次买入不超过计划仓位的50%，价格每下跌10%-15%加仓25%

经典案例极简解读：
2008年金融危机期间，美国银行（Bank of America）股价从每股50美元暴跌至不足3美元，远低于其账面价值约每股20美元。巴菲特以50亿美元获得优先股（年股息6%）+认股权证，到2017年这部分投资价值超过170亿美元，年化回报率超过20%。关键启示：极端恐慌时刻，优质企业的市场价格往往远低于其内在价值。

一句话记忆点：
安全边际不是折扣，而是对"计算错误"和"不可预知风险"的缓冲垫。
```

## Content Quality Guidelines

1. **Data Sources**: SEC filings, company earnings reports, Bloomberg/Reuters
2. **Time Limit**: Use data within 24 hours
3. **No Predictions**: Focus on facts and long-term analysis, not short-term price predictions
4. **Risk Disclosure**: Always include risk warnings
5. **Professional Tone**: Editorial/Magazine style, sophisticated financial terminology
